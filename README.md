# Aritya-arjunan Cmix

## How do I install these formulae?

`brew install aritya-arjunan/cmix/<formula>`

Or `brew tap aritya-arjunan/cmix` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "aritya-arjunan/cmix"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
